## 🎯 What is Pro Ace Tennis?

**Pro Ace Tennis Academy** is a professional, modern website for a tennis academy featuring smooth Lenis scroll animations, responsive design, and a premium dark aesthetic.

---

## ✨ Features

- 🎾 **Modern Dark Design** - Premium sports aesthetic
- 🌊 **Lenis Smooth Scroll** - Buttery smooth scrolling
- 🎨 **Tailwind CSS v4** - Latest styling features
- 📱 **Fully Responsive** - Mobile-first design
- ⚡ **Next.js 16** - App Router + Server Components
- 🔍 **SEO Optimized** - Meta tags and Open Graph
- 🖼️ **Optimized Images** - Next.js Image component
- 🌙 **Dark Theme** - Modern dark mode UI

## 🚀 Quick Start

### Prerequisites

- Node.js 20+ installed
- npm or yarn package manager

### Installation

```bash
# Clone the repository
git clone https://github.com/raghavshahhh/Tennis.git
cd Tennis

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
